PulsarNet Documentation
=====================

Welcome to PulsarNet's documentation. PulsarNet is a robust GUI application designed to streamline the backup processes of various network devices.

Features
--------

* Multi-vendor device backup support (Cisco, Juniper, HP)
* Modern PyQt6-based GUI interface
* Multiple backup protocols (TFTP, SCP, SFTP, FTP)
* Automated scheduling
* Real-time monitoring
* Comprehensive error handling and logging
* Configuration comparison and validation
* Device grouping and batch operations
