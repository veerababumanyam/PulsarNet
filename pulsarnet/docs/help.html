PulsarNet - Network Device Management System

1. Overview

PulsarNet is a comprehensive network device management system designed for efficient backup, monitoring, and configuration of network devices. The application follows ISO/IEC 9126 standards for software quality.

2. Main Features

2.1 Device Management

  • Add, edit, and remove network devices
  • Group devices for organized management
  • Import devices from CSV files
  • Real-time device status monitoring

2.2 Backup Management

  • Manual and automated configuration backups
  • Local and remote storage options (FTP, SFTP, TFTP)
  • Flexible scheduling options (daily, weekly, custom)
  • Retention policy management

4. Storage Configuration

4.1 Local Storage

  • Default location: ~/.pulsarnet/backups
  • Custom path selection available
  • Automatic directory creation

4.2 Remote Storage

  • Supported protocols: FTP, SFTP, TFTP
  • Secure credential management
  • Custom remote path configuration

5. Scheduler

5.1 Schedule Types

  • Daily: Run at specified time every day
  • Weekly: Run on selected days at specified time
  • Custom: Advanced scheduling options

5.2 Schedule Management

  • Add, edit, and remove schedules
  • Device selection for each schedule
  • Status tracking (last run, next run)
  • Automatic execution of due backups

6. Error Handling

  • Comprehensive error logging
  • User-friendly error messages
  • Automatic recovery mechanisms
  • Detailed status reporting
